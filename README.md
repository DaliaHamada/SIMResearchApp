# Workspace

This repository contains the **SIMResearch** iOS demo (Swift + SwiftUI) that collects and displays public device, cellular, and local-network data on a real iPhone.

- **App documentation (overview, data points, API table, limitations, links):** [SIMResearch/README.md](SIMResearch/README.md)  
- **Xcode project:** [SIMResearch/SIMResearch.xcodeproj](SIMResearch/SIMResearch.xcodeproj)  
- **Legacy SIM/SMS R&D write-up (preserved from the original README):** [RESEARCH_NOTES.md](RESEARCH_NOTES.md)  

To run the app, open the Xcode project, set your development team, and deploy to a **physical** device for cellular data (the Simulator has no real SIM information).
